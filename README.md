# Reading Notes

技術書の学習記録リポジトリです。

## 学習スタイル

読んだ内容や感じたことを Claude に話しかけて対話しながら理解を深めます。
会話が一段落したら Claude がノートをドラフトし、リポジトリに記録します。

```
読む → Claude に話しかける → 議論する → ノートをドラフト → commit
```
